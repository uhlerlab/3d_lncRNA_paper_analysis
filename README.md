# 3d_lncRNA_paper_analysis

![Graphical Abstract](graphical_abstract.pdf)

Code to reproduce the analysis for our paper `Tethering by lncRNAs Orchestrates Age-Dependent Interchromosomal Contacts`.

Note that since our analysis uses various genesets, databases, etc, we also provide a `data/` folder with all the cached metadata needed to run our code.

It can be downloaded from this [link](https://drive.google.com/file/d/1gCIpZT9uchl4xYyVFdwuhFwzk7vFFfSY/view?usp=sharing) and added to the main repo as `data/.
